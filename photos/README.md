# 📸 Dossier des photos

Dépose ici les photos qui serviront à générer ta vidéo publicitaire.

## Ordre des plans

Les photos sont utilisées **dans l'ordre alphabétique/numérique naturel**. Pour
contrôler l'ordre, nomme-les avec des numéros :

```
01-produit.jpg
02-detail.jpg
03-ambiance.jpg
10-logo.png
```

(`10` passe bien **après** `02` grâce au tri naturel.)

## Formats acceptés

`.jpg` `.jpeg` `.png` `.webp` `.bmp` `.tif` `.tiff` `.heic`

Tu peux aussi déposer des **clips vidéo** (`.mp4`, `.mov`, `.webm`...) : ils
seront intégrés tels quels dans le montage, entre les photos.

## Conseils

- Utilise des photos de **bonne résolution** (idéalement ≥ 1080 px sur le petit
  côté). Elles seront recadrées pour remplir le format choisi (vertical, carré,
  paysage) — le sujet doit donc rester **centré**.
- Les fichiers de ce dossier ne sont **pas versionnés** dans git (voir
  `.gitignore`), sauf ce README.
